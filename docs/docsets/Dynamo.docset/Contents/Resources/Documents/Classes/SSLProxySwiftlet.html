SSLProxySwiftlet

public class SSLProxySwiftlet: ProxySwiftlet

Swiftlet to allow a DynamoWebServer to act as a https: SSL connection protocol proxy on the same port. This must be come before the DynamoProxySwiftlet in the list of swiftlets for the server for both to work.
